# UI — CITO poc-v-d33

## Layout
- Base em `web/templates/base.html` com navegação fixa.
- Componentes: cards de KPI, tabelas, chips e gráficos.

## Telas
- Doutrina: filtros, KPIs, gráficos e listas de autores, obras e relatores.
- Processos: filtros com select/datalist e KPIs atualizados via `/processos/kpis`.
- Ministros: filtros por ministro/classe/datas e tabela comparativa.
- Scraping: formulário de agendamento, filas, histórico e detalhes de execução.

## Gráficos
- Chart.js para barras, pizza e linha.

## Estilos
- CSS em `web/static/styles.css`.
