# Glossário — CITO poc-v-d33

- `case_query`: coleção de buscas com HTML bruto.
- `case_data`: coleção de processos estruturados.
- `stfDecisionId`: identificador do processo no STF.
- `pipelineStatus`: status consolidado do processamento.
- `caseContent`: conteúdo bruto e limpo do processo.
- `caseData`: dados estruturados extraídos.
- `CITO-REF/1`: protocolo de linhas para referências de notas.
- `CITO-DOCTRINE/1`: protocolo de linhas para doutrina.
- `N/R`: protocolo para legislação (normas e dispositivos).
